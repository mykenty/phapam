## How to use

### Install

```bash
npm i -D @mr-hope/vuepress-plugin-copy-code
```

Or

```bash
yarn add -D @mr-hope/vuepress-plugin-copy-code
```

### Usage

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "@mr-hope/copy-code",
      {
        // your options
      },
    ],
  ],
};
```
