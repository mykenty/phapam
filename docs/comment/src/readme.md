## Install

```bash
npm i -D @mr-hope/vuepress-plugin-comment
```

Or

```bash
yarn add -D @mr-hope/vuepress-plugin-comment
```

### Usage

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "@mr-hope/comment",
      {
        // your options
      },
    ],
  ],
};
```
