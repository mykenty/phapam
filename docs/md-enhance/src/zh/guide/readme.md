# Markdown 中的新语法

通过安装并启用此插件，你可以在 Markdown 中使用更多的语法。

如果你担心你的网站大小，担心使用插件后网站文件大幅增长，那你 duck 不必。如果你不启用某个功能，相关代码则不会包含在输出文件中。
